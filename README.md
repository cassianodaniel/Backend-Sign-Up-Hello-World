# SistemaDeCadastros
Sistema de cadastros desenvolvido com JavaScript, Express, MySQL.

É necessário ter os pacotes:
- MYSQL
- MYSQL2
- SEQUELIZE
- EXPRESS
- EXPRESS-HANDLEBARS
- BODYPARSER
